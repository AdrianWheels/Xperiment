# Prompt para imágenes de gladiadores (robots futuristas)

Quiero sprites individuales para cada clase de gladiador, con estética de robots futuristas. Genera una imagen por gladiador siguiendo estas especificaciones:

- Formato: `PNG` con fondo transparente
- Resolución: `256x256` píxeles por sprite
- Perspectiva: top-down suave (vista cenital con ligera inclinación)
- Iluminación: contraste claro, detalles legibles a tamaño pequeño
- Paleta: neón tecnológico (cian, magenta, amarillo, verde ácido), con metálicos
- Estilo: mecha/robot modular, geometría clara, silueta reconocible
- Variaciones: si aplica, sugiere 3 variantes de casco/armadura
- Exporta con margen consistente y centrado

Para cada gladiador, respeta el tema y habilidades:

1. Critical (Crit): robot asesino preciso, detalles de visor rojo, acentos afilados.
2. Speed: chasis ligero, propulsores visibles, aerodinámico, estelas sutiles.
3. Spinner: núcleo giratorio, anillos concéntricos, sensación de giro y energía.
4. Tank: blindaje pesado, escudo frontal, voluminoso, luces azules de defensa.
5. Spike: púas mecánicas externas, silueta agresiva, reflejos metálicos.
6. Ninja: diseño stealth, placas anguladas, visera oscura, silueta ágil.
7. Prism: superficies prismáticas, refracciones, acentos cristalinos cian/magenta.
8. Orb: esfera gravitacional con anillos, paneles flotantes, emite fuerza.
9. Cube: bloque robusto con motores, caras con biseles, aspecto industrial.
10. Star: núcleo estelar, aletas en forma de estrella, acumuladores de energía.
11. Hex: panales hexagonales, regeneración visual, líneas verdes de vida.
12. Pyramid: pirámide mecánica, modo torreta, base estable, cañones discretos.
13. Bomber: compartimentos de bombas, indicadores de peligro, soporte para huir.
14. Summoner: emisor de drones/pets, antenas, módulo invocador visible.
15. Lancer: lanza/carga frontal, perfil de ariete, luces de carga.
16. Berserker: modo furia, piezas móviles expuestas, rojo intenso en bajo HP.
17. Archer: brazo/cañón de proyectiles, sensores de puntería, silueta equilibrada.
18. Poison: depósitos tóxicos, tuberías verdes, aspecto químico-robot.
19. Illusion: emisor holográfico, clon fantasma, líneas de interferencia.

Entrega cada sprite con nombre de archivo: `<className>_robot_256.png` y opcionalmente una hoja de referencia mínima con tres poses (~512x256) para evaluar silueta y lectura a distancia.
